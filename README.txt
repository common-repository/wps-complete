Wordpress Support Complete brings the tools you are looking for to help maintain and develop your site. Site Backups, Development Tools, Admin menus, API keys to our cloud services. 

== Description ==

This plugin is required for using any of our plugin suite. 
It is the central hub for the extensions available, including an API to hook upto our cloud services. 
More infomation to setup the API can be found <a href="https://wpscomplete.com/wps-complete-api/" target="_blank" >HERE</a>.  
All our plugins can be seen <a href="https://wpscomplete.com/plugins/" target="_blank" >HERE</a>.


== Installation ==

1. Upload WPS complete to wp-content/plugins
1. Click "Activate" in the WordPress plugins menu
1. Select the WPS complete admin navigation to start using
